기본 속성 선택자

        태그에 작성된 특정 속성을 선택하는 선택자

        선택자[속성명="속성값"] { CSS 코드; }

        *"",'' 둘 다 사용 가능

        (참고)
        1) 선택자는 생략할 수 있다.
            -> 생략 시 HTML 문서 내 속성이 일치하는 모든 요소 선택
        2) "속성값"에서 양쪽 쌍따옴표("")는
            홑따옴표('')로 변경하거나 생략도 가능하다.

자식 선택자 (>) 

        지정된 요소 바로 하위(1단계 아래)에 존재하는 
        요소를 모두 선택하는 선택자

        선택자1 > 선택자2{ CSS 코드; }
        - 선택자1 : 부모 요소 선택자
        - 선택자2 : 자식 요소 선택자
